# LeadGen Scout MVP

Инструмент для автоматизированного поиска и квалификации B2B лидов в индустрии HoReCa (рестораны, отели, кафе) с использованием Google Places API.

## Цель проекта
Автоматизировать процесс выявления ресторанов с проблемами в репутационном менеджменте (низкий рейтинг, негативные отзывы без ответа), которые могут быть потенциальными клиентами для маркетинговых услуг.

## Установка

1. **Клонирование репозитория:**
   ```bash
   git clone <repository_url>
   cd LeadGen
   ```

2. **Установка зависимостей:**
   Рекомендуется использовать виртуальное окружение.
   ```bash
   pip install -r requirements.txt
   ```

3. **Настройка окружения:**
   Создайте файл `.env` в корне проекта (на базе `.env.example`):
   ```
   GOOGLE_PLACES_API_KEY=ваш_ключ_api
   DATABASE_URL=sqlite:///leadgen.db
   ```

4. **Инициализация базы данных:**
   ```bash
   python main.py init-db
   ```

## Использование (CLI)

### 1. Поиск лидов
Ищет заведения по координатам, квалифицирует их и сохраняет в БД.
```bash
python main.py search --lat 52.52 --lon 13.405 --radius 5000 --category restaurant
```

### 2. Экспорт данных
Экспортирует найденные лиды в CSV или JSON.
```bash
python main.py export --format csv --status Qualified
```

### 3. Статистика
Выводит общее количество лидов в базе данных по статусам.
```bash
python main.py stats
```

## Структура БД (Lead)
- `name`: Название заведения.
- `address`: Физический адрес.
- `phone`: Контактный телефон.
- `website`: Веб-сайт.
- `rating`: Средний рейтинг Google (3.5 - 4.3 для квалификации).
- `review_count`: Общее количество отзывов (>= 30 для квалификации).
- `last_negative_review_text`: Текст последнего негативного отзыва без ответа от владельца.
- `status`: Статус лида (New, Qualified, Contacted, Rejected).

## Ограничения и Known Issues
- Поиск ограничен лимитами Google Places API (60 результатов за один поиск с использованием пагинации).
- Качество квалификации зависит от наличия текста отзывов в деталях места.
- На текущий момент поддерживается только один негативный отзыв для анализа.
